## Description
```
SabahLabs turns any gathering into a live, fast-paced quiz showdown.

Join a quiz in seconds — no account, no sign-up. Just enter the room code your host shares, pick your nickname, and get ready to compete.

HOW IT WORKS
• The host launches a quiz from the SabahLabs web dashboard
• Players join instantly via the mobile app using a room code
• Each question countdown appears on the host screen
• Tap the coloured tile that matches your answer — fast fingers win bonus points
• A live leaderboard updates after every question

PERFECT FOR
• Classroom lessons and school trivia
• Office team-building events
• Family game nights
• Pub quizzes and social events

NO ACCOUNT NEEDED
Players join with just a nickname and room code. Zero data collected, zero friction.

REAL-TIME ACTION
Powered by live WebSocket connections — answers register instantly and the leaderboard refreshes in real time so everyone feels the excitement.

SabahLabs is free to use. Host unlimited quizzes, invite unlimited players.
```

## Screenshots Required

Apple requires screenshots for at least these two sizes. You can take them using
the iOS Simulator in Xcode, or on a real device.

### Required: iPhone 6.7" (iPhone 15 Pro Max)
- Resolution: **1290 × 2796 px**
- Minimum: 3 screenshots, up to 10

### Required: iPhone 6.5" (iPhone 11 Pro Max / XS Max)
- Resolution: **1242 × 2688 px**
- Minimum: 3 screenshots, up to 10

### Optional but Recommended: iPhone 5.5" (iPhone 8 Plus)
- Resolution: **1242 × 2208 px**

### Suggested Screenshot Scenes (3 minimum)
1. **Landing / Home screen** — showing "Enter Room Code" and "Join Quiz" button
2. **Waiting room** — showing player nickname and "Waiting for host..."
3. **Answer tiles screen** — showing the 4 coloured answer tiles during a live question
4. **Leaderboard** — showing player rankings after a question

### How to take screenshots
```bash
# In Xcode, open Simulator
# Device: iPhone 15 Pro Max (for 6.7")
# Run your app, navigate to each screen, then:
# Simulator menu → File → Save Screenshot  (or Cmd+S)
```

## Review Notes (for Apple reviewer)
```
This app requires a host to create a quiz session via the web dashboard at
https://sabahlabs-frontend.vercel.app before players can join.

To test as a reviewer:
1. Visit https://sabahlabs-frontend.vercel.app
2. Click "Host" and enter admin password: sabahlabs-admin-2024
3. Create a quick quiz or load the sample quiz
4. Start the session — a 6-character room code will appear
5. Open the iOS app, enter the room code and a nickname, tap Join

No account creation is required for players.
```
